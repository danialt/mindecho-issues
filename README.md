# MindEcho Issues

This repository is the public issue tracker for **MindEcho Editor** - an AI-powered text editor.

## About MindEcho

MindEcho is an AI-powered writing assistant that combines:
- Intelligent document editing with AI suggestions
- Real-time collaboration features
- Semantic search across your documents
- Context-aware writing assistance

## How to Use This Repository

This is an **issues-only repository**. The source code is maintained in a private repository.

### Reporting Issues

Please use the [Issues tab](../../issues) to:
- Report bugs
- Request features
- Ask questions
- Provide feedback

### Before Submitting

1. Search existing issues to avoid duplicates
2. Use clear, descriptive titles
3. Provide steps to reproduce (for bugs)
4. Include screenshots if applicable
